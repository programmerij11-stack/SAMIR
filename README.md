## Prototype de transport du personnel

Le prototype autonome **NavetteOps** est disponible dans
[`transport-personnel/`](transport-personnel/).

Ouvrir `transport-personnel/index.html` via un serveur HTTP statique, par
exemple :

```bash
python3 -m http.server 8000
```

Puis visiter `/transport-personnel/`.
